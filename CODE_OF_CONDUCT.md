# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

We are committed to a harassment-free, welcoming experience for everyone. Be respectful, assume good
faith, and focus on what's best for the community and for the integrity of a system people rely on to
run fair elections. Harassment, personal attacks, and publishing others' private information are not
tolerated. Maintainers may remove contributions that violate this code and may ban offenders.

**Reporting:** **info@ip21.si** (handled confidentially).

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
